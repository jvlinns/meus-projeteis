Aula de hoje: CSS

Existem 3 tipos de incorporações do CSS:

CSS in line style: Bom para alterar pontualmente um local, mas polui muito o código. É a pior de todas, não usar.

CSS internal Style: Atribui os códigos no início da pg: Deixa o HTML extenso mas agrupa o codigo do estilo em um só lugar. Use essa técnica quando for criar páginas isoladas com estilos próprios, que não serão replicados em outras páginas

CSS External Style: Atribui o código em um arquivo .css externo. É a melhor opção para seu código, deve ser usada quando se tem várias páginas, ela organiza melhor seu código
